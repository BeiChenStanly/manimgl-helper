## Clone the manimgl Repository

To get the **latest development version** of manimgl (recommended), clone the official repository from GitHub:

```bash
git clone https://github.com/3b1b/manim.git
cd manim
```

This gives you the most up-to-date version with all the latest features and bug fixes.

> **Why git instead of pip?** The pip package (`manimgl`) may lag behind the GitHub repository. The git version ensures you have the latest rendering improvements and the full `checkpoint_paste()` workflow.
