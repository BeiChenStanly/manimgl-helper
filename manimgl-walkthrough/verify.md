## Verify Installation

Run the **Check manimgl Installation** command to verify everything is set up correctly:

- Open the Command Palette (`Ctrl+Shift+P`)
- Run **ManimGL: Check manimgl Installation**

Or click the manimgl version indicator in the status bar.

You should see the installed version number. If you see "manimgl not found", double-check that:

1. The correct Python interpreter is selected in VS Code
2. `pip install -e .` completed without errors
3. `manimgl --version` works in your terminal
